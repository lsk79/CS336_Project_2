  Input: "In a Python script running on a Linux server (ILab), how do I connect to a PostgreSQL database using psycopg2, prompt for credentials using getpass, execute a SELECT query provided as part of the command line argument, and format the output results for display?"
  Output: "Here’s a clean way to do exactly what you’re describing on a Linux server (like iLab): connect to PostgreSQL with psycopg2, securely prompt for credentials using getpass, take a SQL SELECT query from the command line, execute it, and print formatted results.

import psycopg2
import sys
from getpass import getpass

def main():
    if len(sys.argv) < 2:
        print("Usage: python script.py \"SELECT * FROM table;\"")
        sys.exit(1)

    query = sys.argv[1]

    # Secure credential input
    db_name = input("Database name: ")
    user = input("Username: ")
    password = getpass("Password: ")

    try:
        conn = psycopg2.connect(
            dbname=db_name,
            user=user,
            password=password,
        )

        cur = conn.cursor()
        cur.execute(query)
        conn.commit()
        cur.close()
        conn.close()

    except psycopg2.Error as e:
        print("Database error:", e)

if __name__ == "__main__":
    main()
